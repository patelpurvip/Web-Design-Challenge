Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes in relation to distance from the Earth’s equator. The analysis entailed pulling data from the OpenWeatherMap API to assemble a dataset of over 500 cities. The data presented reflects weather patterns reported on March 28, 2020.

After assembling the data set, Matplotlib was used to plot various comparisons of weather vs. latitude to evaluate possible correlations within the data. Factors evaluated included temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
